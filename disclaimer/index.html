Disclaimer

This website is an unofficial archive built from automated processing of public Tim Cain videos.

The transcript text on this site is generated from subtitle files and then cleaned with a mixture of heuristics and AI-assisted review. Even after cleanup, it can still contain recognition mistakes, punctuation problems, missing words, or incorrect proper nouns.

Nothing on this site should be treated as an official transcript or an official publication from Tim Cain. The authoritative source is always the original video on YouTube.

What this archive is for

  • easier text browsing
  • tag-based navigation across topics
  • internal linking between related videos
  • long-form reading when watching a video is inconvenient

What this archive is not

  • an official site
  • a perfect transcript source
  • a replacement for the original video context

How the pipeline works

  1. Once a day, GitHub Actions checks the channel feed for newly public videos and updates the local metadata in this archive.
  2. If subtitles are available, the pipeline downloads the English VTT caption file for the video.
  3. That subtitle file is converted into plain text, punctuated, cleaned up, and split into readable paragraphs.
  4. References to other videos mentioned in descriptions are extracted and linked back into the archive.
  5. Tags are assigned from a fixed tag catalog, using the title, description, transcript paragraphs, and local embedding-based matching.
  6. The result is written into the repository as structured metadata, subtitle files, and a Markdown transcript page.
  7. Metadata is validated before the updated corpus is published.
  8. A separate workflow exports the corpus into Hugo content, rebuilds the static site, and publishes the generated files into a public site repository.

What can still go wrong

  • subtitles may already be incomplete or wrong before any cleanup starts
  • punctuation and paragraph boundaries can still be imperfect
  • proper nouns can still be misheard
  • tags can be incomplete, overly broad, or occasionally wrong

Source of truth

If you want the original source material, use the embedded video or the direct YouTube link on each video page.
